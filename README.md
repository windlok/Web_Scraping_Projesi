# Web Scraping Projesi

Bu proje, veri kazıma (web scraping) çalışmaları için hazırlanmış bir Jupyter Notebook tabanlı örnektir. Proje, farklı web kaynaklarından veri çekmek, temizlemek ve temel analiz hazırlıkları yapmak için kullanılabilir.

## İçindekiler
- `Web_Scraping_Projesi.ipynb` — Ana çalışma not defteri.

## Gereksinimler
- Python 3.8 veya üzeri
- Jupyter Notebook veya JupyterLab
- Yaygın kullanılan kütüphaneler: `requests`, `beautifulsoup4`, `pandas`, `lxml`


## Kullanım
- Notebook içindeki hücreleri sırayla çalıştırın. Her bölümde veri çekme, parse etme ve temel temizleme adımları bulunmaktadır.
- İsterseniz kod bloklarını modüler bir .py dosyasına taşıyıp bir komut satırı arayüzü ile otomatikleştirebilirsiniz.

## Proje Yapısı
- `Web_Scraping_Projesi.ipynb` — Analiz ve scraping akışı.
